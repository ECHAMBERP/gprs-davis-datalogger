### Ultimate GPS V3 (http://www.adafruit.com) interface library for Arduino

This library uses the hardware serial port for the communication with the GPS module and requires 3 pins 
on the Arduino (pin 0, pin 1 and another digital pin for module power control).

It can be used to :

- acquire the current position (with timeout and HDOP limit specified by user)
- acquire the current date and time

Dependencies : SoftwareSerial library (for debugging purpose only).
