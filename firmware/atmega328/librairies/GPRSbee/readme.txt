### GPRSBEE modem (http://www.gprsbee.com) interface library for Arduino.


This library uses the softwareSerial library for the communication with the modem and requires 
4 digital pins on the Arduino (TX, RX, power control and power status monitoring).

It can be used, among other, to enable interactions between the Arduino and web servers via 
HTTP GET and POST requests. 


Dependencies : SoftwareSerial library.
